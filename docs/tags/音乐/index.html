西北民大乐队 DoomWay黯淡方式将在12月24日于葵艺术空间进行演出

Doom Way（黯淡方式）

  由几位相同而又不同的大学生组建于22年3月，风格主要为后朋克和情绪朋克，现有原创七首，已发布两首。低声诉说是他们的表达方式。信奉极简主义，坚信Ian Crutis无时无刻不在紧握他们的手，以此来创作后朋克音乐。歇斯底里的爆发也是他们的表达方式。受第二波以及三波emo浪潮的强烈感召，任由破碎的心情以情绪硬核作为载体而疯狂发泄。
